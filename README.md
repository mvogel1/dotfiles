# .dotfiles

## Homebrew Usage

### Utilities

#### Search

```
brew search [name]
```

#### Get Info
```
brew info [name]
```

#### Install
```
brew install [name]
```


### Desktop Apps (Cask)

```
brew search --cask [name]
```

#### Get Info
```
brew info --cask [name]
```

#### Install
```
brew install --cask [name]
```
